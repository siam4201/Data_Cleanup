# Data_Cleanup

- Data cleanup and visualization techniques applied on cricket  plays dataset from espn.
- For cleanup we used IQR to find outliers.
- Handled missing values by using mean or median depending on if there are outliers.
- Corrected data types.
- Split and dropped columns to have a cleaner dataset.
- Removed ambiguos symbols.
